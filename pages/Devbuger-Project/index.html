DevBurger

O DevBurger é um projeto de um sistema de pedidos de lanches, desenvolvido como parte do curso de Desenvolvimento Web Full Stack do DevClub. O objetivo principal deste projeto é criar uma aplicação web que permita aos usuários fazerem pedidos de lanches de forma fácil e rápida, além de proporcionar um painel administrativo para gerenciamento de pedidos, estoque e relatórios.

Visão Geral do Projeto

O DevBurger consiste em duas aplicações principais:

1. Frontend para Clientes: Uma interface amigável onde os clientes podem visualizar o cardápio, selecionar produtos, personalizar seus pedidos e realizar o pagamento.

2. Painel Administrativo: Uma interface para gerentes e funcionários acompanharem pedidos em tempo real, gerenciarem o cardápio, controlarem o estoque e acessarem relatórios de vendas.

Stack Tecnológica

Frontend

React.js para construção da interface do usuário, com React Router para navegação e Styled Components para estilização. Implementamos também Redux para gerenciamento de estado global da aplicação.

Backend

Node.js com Express para a criação da API RESTful, Sequelize como ORM para interação com o banco de dados, e autenticação JWT para segurança.

Banco de Dados

PostgreSQL para armazenamento de dados relacionais como usuários, produtos, pedidos e transações. Utilizamos MongoDB para logs e dados não estruturados.
